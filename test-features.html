Admin panel opens as a sidebar from the right

Toggle switch in admin settings to enable/disable moderation
When disabled: Words auto-approve without manual review
When enabled: Words go to pending queue for approval
Setting persists across sessions
✓ Success: Admin can toggle moderation on/off. This allows for different use cases - moderated sessions or open participation.

3. Light/Dark Mode IMPLEMENTED

Theme toggle button in header
Dark mode with appropriate color schemes
Follow system preference option
Smooth theme transitions
Theme preference saved in localStorage
✓ Success: Users can switch between light and dark themes. Theme preference is remembered across visits.
